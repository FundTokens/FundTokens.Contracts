# FundTokens.Contracts

FundTokens lets anyone hold a trustless, self-custodial basket of BitcoinCash native assets using CashTokens and smart contracts. No middlemen. Transparent rules. On-chain

## v0.0.1 - Initial [WARNING: Double Minting]

* Blaze Hackathon contract
* Use NFT for FundToken
* Burns tokens for redemption
* Hardcoded to two assets

## v0.0.2 - Fungible Tokens

* Designed with fungible tokens in mind
* Reduced contract count
* Maintains two assets hardcoded

## v0.0.3 - Dynamic Asset Count

* Support a dynamic asset set (>=1)
